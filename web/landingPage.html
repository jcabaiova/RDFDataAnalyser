How it Works?

Specify domains

Look at domains and entities and add new ones on which your dataset will be calculated. You can insert more domain en entities together with the help of import file in format N-Triples or entity by entity.

Upload dataset

Import your RDF data to our data analysis software. Avaliable formats are SPARQL endpoint or file in HDT format. Put name of dataset, path to dataset, ontology predicate and chose for the type of calculation.

View a result

Wait a few minutes and after successful finish of the analysis you can view your result in form of chart and table. Click on the part of chart and you get to another level. You can export result in table too.

About It

This application was created as the result of my master thesis work with name "Sumarizing Linked Open Data Datasets". The main method of this application is calculation of domain representation for Linked Open Data dataset. In the application are available this basic functions:

  • Defining domains and entities for calculation
  • Importing dataset in SPARQL endpoint or HDT format
  • Macro visualization - dataset coverage for each defined domain shown in the chart and micro visualization - completeness of information for a given entity type shown in the table
  • Dataset comparison
This application should be useful mainly for these, who need to find out domain representation of their Linked Open data dataset or they need to compare twodifferent datasets based on the domain specification.
